# Advanced-SQL-Exercises

## NULL & DATE Functions

### Overview
This repository contains advanced SQL exercises focused on:
- Handling NULL values
- Working with DATE and TIME functions

These exercises are designed to build practical skills for data analysis, reporting, and database management.


### Topics Covered

#### NULL Handling
- `IS NULL` / `IS NOT NULL`
- `COALESCE()`
- `NULLIF()`
- `IFNULL()` / `NVL()`

Key Concepts:
- Managing missing data  
- Preventing calculation errors  
- Replacing NULL values effectively  

---

#### DATE Functions
- `DATEADD()`
- `DATEDIFF()`
- `CURRENT_DATE`, `CURRENT_TIMESTAMP`
- `EXTRACT()` / `DATE_PART()`
- `TO_CHAR()` (format dates)
- `TO_DATE()` (convert strings)
- `DATEFROMPARTS()`

Key Concepts:
- Date calculations  
- Time differences  
- Formatting output  
- Extracting date components  


### Exercises Included
- Adding and subtracting dates  
- Calculating time intervals  
- Formatting timestamps  
- Handling NULL values in queries  
- Real-world SQL scenarios  


### Learning Outcomes
- Handle NULL values confidently  
- Perform advanced date calculations  
- Format and manipulate timestamps  
- Write efficient SQL queries for real-world use
